# India-weather-forecast
Real-time weather forecasting web app for Indian states and districts using Weather API.
Built with modern JAMstack architecture (JavaScript, APIs, Markup) to ensure high performance, scalability, and security.
Uses client-side JavaScript to fetch and display real-time weather updates dynamically from reliable external APIs.
Provides accurate and up-to-date forecasts for Indian locations with fast API response handling.
Pre-rendered static markup improves loading speed, SEO ranking, and protection from server-side vulnerabilities.
Optimized for mobile responsiveness, smooth user experience, and lightweight performance across devices.
Deployed on a global CDN-based static hosting platform for maximum reliability, speed, and easy maintenance.
